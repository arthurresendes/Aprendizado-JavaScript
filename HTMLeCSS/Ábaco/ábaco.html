Ábaco -

Esta API permite que você crie contadores numéricos simples. IaaS, Integer as a Service.

Tudo se resume a:

• Criar e incrementar contador

• Ver dados do contador

Todos os contadores são “não listados” — acessíveis se você souber a chave, mas não listados publicamente em nenhum lugar.
